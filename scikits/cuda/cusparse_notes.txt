Python wrappers exist for all 330 individual functions in cuSPARSE v6.5.  These get automatically generated from the cusparse_v2.h/cusparse.h headers upon the first import of scikits.cuda.cusparse.  The code performing this autogeneration is within _cusparse_cffi.py & _cusparse_cffi_autogen.py
cusparse.py contains higher level data-type agnostic wrappers to a subset of the functions I was interested in using in the near-term.  These are primarily CSR matrix-vector and matrix-matrix multiplication and some basic format conversions.

Specifically the following have simplified interfeaces within cusparse.py

Level2 functions:
csrmv

Level3 functions:
csrmm
csrmm2

extra functions:
csrgeam
csrgemm

conversions:
csc2coo
csc2dense
csr2coo
csr2csc
csr2dense
coo2csc
coo2csr
dense2csc
dense2csr
nnz

There is also a CSR class that incorporates much of this functionality in an even simpler manner.
